# 🏪 Anker Store 出店分析ダッシュボード

重回帰分析を用いた店舗売上予測と新規出店シミュレーションシステム

## 📊 機能

### 1. 分析結果
- 重回帰分析の結果表示
- 各変数の影響度（標準化係数）
- 決定係数・F統計量などの統計指標
- 実測値 vs 予測値の散布図

### 2. 新規出店シミュレーター
- 立地条件を入力して売上予測
- 類似既存店舗の表示
- 平均との比較

### 3. 既存店ポテンシャル評価
- ポテンシャル高店舗（過小評価）の特定
- 要改善店舗（過大評価）の特定
- 店舗別の残差分析

### 4. 感度分析
- 各変数の変化が売上に与える影響
- 店舗別シミュレーション

## 🚀 ローカル起動方法

### 必要環境
- Python 3.9以上
- pip

### インストール

```bash
cd store_analytics_dashboard
pip install -r requirements.txt
```

### 起動

```bash
streamlit run app.py
```

ブラウザで自動的に開きます。手動の場合: http://localhost:8501

## 📈 分析概要

### 目的変数
- 平均売上(税抜)

### 説明変数（18変数）
1. 館売上規模 (1-5)
2. 自店坪数
3. 間口スコア (1-5)
4. KDDI人流数値
5. リピーター率
6. フロア評価 (0.5/0.8/1.0)
7. 視認性 (1-5)
8. 動線スコア (1-5)
9. 郊外商業 (0/1)
10. アウトレット (0/1)
11. 駅ビル駅近 (0/1)
12. 量販店内 (0/1)
13. 路面店 (0/1)
14. 量販店近接 (0/1)
15. 商圏年収 (1-5)
16. 競合店数 (0-5)
17. Cafe (0/1)
18. インバウンド (1-5)

### 分析手法
- 重回帰分析（OLS）
- 標準化係数による影響度評価
- VIF（多重共線性）チェック

## 📝 使い方

### 新規出店シミュレーション
1. 「🔮 新規出店シミュレーター」を選択
2. 立地条件（坪数、人流、立地タイプ等）を入力
3. 予測売上が即座に表示

### 既存店のポテンシャル確認
1. 「🏬 既存店ポテンシャル」を選択
2. 色分けされた散布図で一目で確認
3. タブで詳細データを表示

### 感度分析
1. 「📈 感度分析」を選択
2. 分析する変数と基準店舗を選択
3. グラフで影響度を可視化
